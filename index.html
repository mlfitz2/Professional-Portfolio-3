PubCrawler

PubCrawler allows users to select a city, or use their current location, to see a map of breweries in the area. They can select the ones they like, and the app will map out the most efficient route for a pub crawl between them. The user can save their favorite routes to local storage to refer back to them next time.

Workday Planner

This workday planner allows the user to input their plans for a day into one-hour time blocks. The data is saved to local storage to be available at a later time. The color coding provides an at-a-glance view of past, present and future activities.

Interactive Quiz

This quiz helps the user practice their knowledge of JavaScript by testing their understanding of key concepts. The user attempts to answer all questions correctly as fast as they can. Incorrect answers are penalized with a reduction on the timer, and the final score is the remaining time. High scores can be saved to a scoreboard on local storage.
